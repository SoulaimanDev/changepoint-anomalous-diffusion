# Changepoint Detection in Synthetic Anomalous Diffusion Trajectories

## Project Overview

This repository presents a Master’s final project on changepoint analysis in synthetic anomalous diffusion trajectories. The objective is to study how deep learning models can identify transitions between diffusion regimes and estimate the temporal position at which these transitions occur.

Anomalous diffusion is commonly used to describe systems whose motion deviates from classical Brownian behavior. In this project, trajectories are generated from different anomalous diffusion models and combined to create controlled changepoint scenarios. The repository provides datasets, notebooks, and model evaluations for studying changepoint localization and binary changepoint detection under synthetic experimental conditions.

## Research Workflow

The project is organized into two experimental phases.

### Phase 1: Changepoint Localization

In the first phase, every trajectory contains one changepoint. The task is therefore restricted to temporal localization: the model must estimate the position of the changepoint in a trajectory of fixed length `L = 100`.

This phase evaluates how different neural architectures localize transitions when the presence of a changepoint is guaranteed.

### Phase 2: Binary Detection and Temporal Localization

In the second phase, the dataset includes trajectories with and without a changepoint. The problem is formulated as a dual task:

1. Binary detection: determine whether a changepoint is present.
2. Temporal localization: estimate the changepoint position when a changepoint exists.

The main input representation in this phase is the increment signal `dx(t)`, rather than the raw position trajectory `x(t)`. This representation emphasizes local variations in the trajectory and provides a direct description of short-term dynamical changes.

## Dataset Description

The synthetic trajectories have fixed length:

- `L = 100`

The diffusion models considered in the dataset are:

- Annealed Transient Time Motion (`ATTM`)
- Continuous Time Random Walk (`CTRW`)
- Fractional Brownian Motion (`FBM`)
- Lévy Walk (`LW`)
- Scaled Brownian Motion (`SBM`)

For changepoint trajectories, each sample is built from two segments generated by different diffusion models. Ordered transitions are considered, which means that a transition from model A to model B is treated as distinct from a transition from model B to model A.

The datasets are designed to remain balanced across diffusion classes and transition types. In Phase 1, all samples contain a changepoint. In Phase 2, the dataset includes both trajectories with changepoints and trajectories without changepoints, allowing the models to be evaluated on binary detection and localization simultaneously.

## Models

### LSTM

The LSTM model is used as a recurrent baseline for sequential trajectory analysis. It processes the temporal structure of the input signal and captures dependencies across the trajectory. In this project, it provides a reference point for evaluating whether recurrent memory alone is sufficient for changepoint localization and binary detection.

### ConvLSTM

The ConvLSTM architecture combines local pattern extraction with recurrent temporal modeling. The convolutional component helps identify short-range changes in the signal, while the recurrent component models the temporal evolution of the trajectory. This structure is especially relevant when local variations in `dx(t)` carry information about regime transitions.

### Transformer Encoder

The Transformer Encoder relies on attention mechanisms to model relationships between different temporal positions. Instead of processing the sequence only through recurrence, it can compare distant parts of the trajectory directly. This makes it suitable for studying whether global temporal dependencies improve changepoint detection and localization.

### ConvTransformer

The ConvTransformer combines one-dimensional convolutional layers with attention-based temporal modeling. The convolutional stage extracts local features from the increment signal, while the Transformer component models broader temporal interactions. This hybrid design aims to benefit from both local sensitivity and long-range sequence representation.

## Phase 1 Results

Phase 1 evaluates temporal localization only, because all test trajectories contain a changepoint. The reported metrics are the mean absolute error (`MAE`) and root mean squared error (`RMSE`) between the predicted and true changepoint positions.

| Model | MAE | RMSE |
|---|---:|---:|
| ConvTransformer | 7.20 | 11.63 |
| LSTM | 7.72 | 11.71 |
| Transformer | 7.87 | 12.37 |
| ConvLSTM | 8.71 | 12.58 |

The results indicate that the ConvTransformer provides the lowest localization error in this first experimental setting, while the LSTM and Transformer models remain competitive. The ConvLSTM shows a higher localization error in this phase.

## Phase 2 Results

Phase 2 evaluates both binary changepoint detection and temporal localization. The threshold is selected for the binary decision task. The localization metrics `MAE` and `RMSE` refer to the error on trajectories that contain a changepoint.

| Model | Threshold | Accuracy | Precision | Recall | F1-score | FPR | FNR | MAE | RMSE |
|---|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| LSTM | 0.35 | 0.732765 | 0.734069 | 0.729980 | 0.732019 | 0.264450 | 0.270020 | 12.733092 | 16.350369 |
| ConvLSTM | 0.25 | 0.789130 | 0.797980 | 0.774280 | 0.785951 | 0.196020 | 0.225720 | 7.664480 | 11.918347 |
| Transformer Encoder | 0.30 | 0.771165 | 0.809284 | 0.709540 | 0.756137 | 0.167210 | 0.290460 | 8.741436 | 12.530686 |
| ConvTransformer | 0.30 | 0.803675 | 0.818374 | 0.780590 | 0.799036 | 0.173240 | 0.219410 | 7.536969 | 11.570613 |

The ConvTransformer achieves the strongest overall performance in Phase 2, with the highest accuracy and F1-score among the evaluated models. It also obtains the lowest localization error in terms of both `MAE` and `RMSE`.

## Main Findings

- The ConvTransformer provides the best overall performance in the second experimental phase, combining strong binary detection with accurate temporal localization.
- The ConvLSTM improves localization compared with the LSTM baseline, suggesting that local feature extraction is useful for detecting regime transitions in `dx(t)`.
- The Transformer Encoder obtains strong precision, but its lower recall indicates that some true changepoints remain more difficult to detect.
- The LSTM remains a useful recurrent baseline, although its localization error is higher in Phase 2.
- The increment representation `dx(t)` helps describe local trajectory variations and supports the detection of changes in dynamical behavior.
- Transition-wise analysis remains important because changepoint difficulty depends on the pair of diffusion models involved in the transition.
